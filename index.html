フィジーク トス

「フィジークの選手はサーフパンツを履く」。この一文は「柔道の選手は道着を着なければならない」という競技におけるルールを示しているだけではない。スポーツの大会において多くの場合、着用する衣類には規定が存在する。そのほとんどが選手たちの能力を純粋に見極めるために、その外的な要因となる部分をできる限り均質化しようとするものであろう。フィジーク選手のサーフパンツはそうした規定であると同時に、競技の審査に作用するものである（ナイストスはプレイに影響する）。どうしようもなくある規定や約束事はフォーマットと言ってみてもいいかもしれない。そんなフォーマットに対して「のる」でも「おりる」でもなく「つかってみる」。

接続

ぜんぜん別の場所にあるものが、重なって見える。いま起こっていることが、いつか体験したことのように感じられる。

インターネット = 現代美術 = 映画 = 演劇 = 詩 = YouTube = 無印良品。
時間や空間を超えて別々のものが重なって見えてしまうことが、(インターネット|現代美術|映画|演劇|詩|YouTube|無印良品)だと思う。

会社員

マイノリティな人たちやオルタナティブな選択肢は、はじめからマイノリティやオルタナティブとして生まれたわけではなく、認知された結果として母数があきらかになったことで、マイノリティやオルタナティブになった。メジャーとは、多数派ではなく名前がついていない状態で、つまり「その他」である。

じぶんが分類されている区分のなかから、マイノリティでオルタナティブなものを探す。男性、20代 = 平成生まれ、神奈川県 = 日本 = 日本人、会社員。

会社員を、マイノリティで、オルタナティブな存在だと考える。

美術大学で表現を学んだ人たちが、会社員としてウイルスのように社会へ入り込んでいく。美術大学は、社会をよりよくするためのアクション。表現は教えることができるのかとか、もはやそういう話ではない。

会社で働く人たちが、仕事によって、ウイルスのように世界へ接続していく。
